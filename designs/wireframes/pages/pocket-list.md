# 口袋名單 / 收藏 · Pocket List

## 目的

「咖啡口袋」的命名由來就是這頁:**使用者親手挑出來、會反覆造訪的店**。
不是 Google Maps 那種一鍵收藏到死的星號清單,而是分類化、情境化的小口袋。

## 入口

- 底部分頁 (Tab Bar) 第二個圖示 · 標題「口袋」
- 個人頁的「我的收藏」入口

## 互動

- **新增口袋**:輸入名字 (e.g. 「工作」「跟朋友」「下雨天」) + 選 emoji / 顏色,可選「公開分享」
- **長按店家**:出現選單 → 移到其他口袋 / 移除 / 編輯備註
- **左滑店家**:快速移除 (帶 undo toast,5 秒)
- **點店家**:進入 [cafe-detail](cafe-detail.md)
- **「在地圖上看這個口袋」**:跳到 Mobile / Desktop 地圖頁,只渲染此口袋的 pin,標題改為「口袋 · 工作」

## 排序

預設依「加入時間 (新 → 舊)」。可切:
- 距離 (需定位)
- 名字
- 上次造訪 (未實作 v1)

## 邊界狀態

- 完全沒收藏 → 大留白 + 手繪箭頭指向地圖頁的 ♡ 圖示 + 文案「在店家頁點 ♡ 加入口袋」
- 該口袋是空的 → 顯示 emoji + 「這個口袋還是空的,去逛逛吧 →」
- 未登入 → 整頁鎖,卡片堆疊的 placeholder + 「登入後可同步你的口袋」按鈕

## 同步行為

- 已登入: Supabase 即時同步
- 未登入: 暫存 localStorage,登入後自動 merge (相同店家以「加入時間較早」為準)
