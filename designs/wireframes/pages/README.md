# Coffee Pocket — 頁面描述

文字版的 wireframe 描述,補足目前 jsx wireframe 尚未涵蓋的頁面。
每份檔案描述一個頁面的目的、版面、主要元件、互動與狀態。

風格沿用 [specs/requirements.md](../../../specs/requirements.md):
扁平化、低視覺壓力、分割線優先於陰影、米白紙感配色、情境式搜尋為核心。

## 既有 wireframe (jsx)

- `wf-home.jsx` — 首頁 / 搜尋入口 (4 variations)
- `wf-mobile.jsx` — Mobile 地圖 + 結果頁 (4 variations)
- `wf-desktop.jsx` — Desktop 地圖 + 結果頁 (4 variations)
