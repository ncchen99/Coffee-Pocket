# 設定 · Settings

## 目的

放少數真的需要使用者決定的選項。
**不要**塞通知、行銷、A/B、莫名其妙的開關。每個選項都該有非預設值的清楚理由。

## 入口

- [profile](profile.md) 的「⚙ 設定」

## 互動

- 變更立即生效,不需「儲存」(無 dirty state 概念)
- 「刪除帳號」:二次確認 + 文字輸入「DELETE」才執行;真的會刪除 (符合 GDPR / 個資法)
- 「匯出我的口袋」:下載 JSON,包含口袋名稱、店家 Place ID、自訂備註、加入時間

## 邊界狀態

- 未登入 → 整個「帳號」區塊改為一個「登入」按鈕
- 系統不支援 dark mode 偵測 → 「系統」選項灰掉,預設改為「淺」
