# 個人頁 · Profile

## 目的

讓使用者看見:
1. 自己累積的口袋 (數量、店家)
2. 自己對社群的貢獻 (新增 / 修正 / 投票)
3. 設定入口

不是「個人化推薦的展示頁」(那是首頁的責任)。

## 入口

- 底部分頁 Tab Bar 第三個「我」
- 各頁右上角頭像點擊

## 互動

- 數字區塊點擊各跳對應頁面 (收藏 → [pocket-list](pocket-list.md))
- 貢獻列表項目點擊 → 對應的店家頁,並 highlight 自己改過的欄位
- 「反饋」開啟 mailto: / 內建表單

## 邊界狀態

- 未登入: 直接顯示登入 CTA + 為什麼登入的好處 (與 [login.md](login.md) 相同文案)
- 剛註冊、無貢獻:「我的貢獻」區改為引導文案「修正一個錯誤,讓臺南的咖啡資料更準 →」連到隨機一間資料較舊的店

## Desktop 差異
