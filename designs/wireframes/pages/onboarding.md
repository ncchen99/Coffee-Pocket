# 第一次開啟引導 · Onboarding

## 目的

只做兩件事:
1. **建立心智模型**:這不是 Google Maps,是「情境搜尋」
2. **拿到位置權限** (App 一切體驗的基礎)

不做帳號註冊,不做興趣偏好調查 (那留給第一次互動觀察)。
三頁,可跳過。

## 版面


## 互動

- 每頁可橫滑 / 點下一步,也可右上「跳過」
- Page 3 的「開啟定位」觸發系統權限對話框;授權後 → 直接進首頁;拒絕 → 跳到「手動指定地區」 picker
- 完成後寫一筆 `onboarded_at` 到 localStorage,**再不顯示**
- 全程不要求登入

## 邊界狀態

- 已 onboarded 過 (localStorage 有紀錄) → 直接進首頁
- App 更新有重大新功能 → 新版可選擇性顯示一個 1-page「what's new」(不放在 onboarding 流程中)
- 拒絕定位 + 沒手動指定 → 預設「臺南中西區」,並在首頁頂部 banner 提示「你正在以中西區為中心,要改 →」
