# 進階篩選頁 · Advanced Filter

## 目的

把「多標籤交叉篩選」做成主功能,而非附屬抽屜。
使用者在這頁可以:**時間 + 距離 + 人數 + N 個標籤** 一次決定。
**底部即時筆數**永遠存在,讓使用者邊調邊知道剩幾間。

## 入口

- 應該會是在檢視結果的頁面這邊，可以有進階搜尋
- Mobile 結果頁的 ⚙ 篩選按鈕 → 全螢幕推上來
- Desktop 結果頁的左側 aside 已內嵌簡易版,此頁是「完整版」的 modal

## Desktop 差異

- 不必是全螢幕 modal,**直接內嵌在左側 aside (340 寬)**,捲動即可
- 即時筆數位於 aside 底部 + 結果列表頂部都各有一份
- 多了一個「儲存此組篩選為情境」的小按鈕 (進階使用者)

## 互動

- 每次切換任一條件 → **debounce 300ms 後重新查筆數**,footer 數字動畫更新
- 筆數 0 → footer 改紅字 + 顯示「沒有符合的店,試試 [移除安靜] 或 [距離拉到 5k]」(可點建議)
- 「重置」清空所有條件,回到預設「現在 / 3km / 不限人數」
- 「✕」關閉,**不套用** (草稿狀態丟棄);「顯示 N 間 →」才套用

## 預設值

- 時間: 現在
- 距離: 3km
- 人數: 不限
- 標籤: 全空
- 排序: 距離

## 邊界狀態

- 未授權位置 → 距離 slider 灰掉 + 一行提示「開啟定位才能依距離排序」
- 篩選條件衝突 (例:同時要求「安靜」和「熱鬧」) → 該標籤群組改為 radio 不可同選
